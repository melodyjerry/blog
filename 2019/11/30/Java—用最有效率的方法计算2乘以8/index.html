最有效率的方法 👉 2 << 3

其中，左移3位相当于乘以2的3次方，右移3位相当于除以2的3次方

再比如，31 * num

等价于( num << 5 ) – num

左移5位相当于乘以2的5次方（32），再减去自身（-1），这一过程就相当于乘以31
